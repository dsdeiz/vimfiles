# Notes

* Create the directories `undo`, `backup`, and `swap` in `$HOME/.vim/tmp`.
* Install `ctags-trunk` (or [`universal-ctags`] [3] - `brew install universal-ctags`). See this [section] [1].
* Install `ag`. See this [section] [2].
